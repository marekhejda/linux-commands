Linux commands

Standard commandsDescription
pwdPrint working directory
whoamiDisplay username of currently logged user
clearClear terminal window
historyShow command history
exitExits terminal
echoDisplay string into terminal windows
man COMMAND, COMMAND --helpdisplay manual or help for command
Directory and file commandsAttributesDescription
cd ../BACK_FOLDER, cd FORWARD_FOLDER, cd /ABSOLUTE_PATH---Change directory
mkdir FOLDER_NAMEmkdir -p PATH/FOLDER_NAMECreate new folder where specified
rmdir FOLDER_NAME_TO_DELrmdir -p FOLDER_PARENT_TO_DEL/FOLDER_NAME_TO_DELRemoves empty folder or parent of folder
lsls -l, ls -h, ls -a, ls -t, ls -rShow list of files in the current directory, where -l is advanced list, -h is human readable size format, -a shows all entries, -t sort files by date and -r shows list in reverse order
treetree -dShow tree of child directories and files, or with -d only child directories, tree command needs to be installed by: yum install tree
cp SOURCE DESTINATIONcp -r SOURCE_NON_EMTY_DIRECTORY DESTINATIONCopy source files and folders into destination path
mv SOURCE DESTINATION---Move or rename source file to destination path
rm FILENAMErm -rf FILE_OR_NON_EMPTY_FOLDERRemove file or folder, - means forced delete of every file and -r for recursive delete of each child and parent directories
touch FILE_NAMEtouch -d YYYY-MM-DD FILE_NAMECreate new empty file or change timestamp of file with -d
tail FILENAMEtail -n5 FILENAMEDisplay last 10 rows of file specified, or custom nuber of rows with -n#
head FILENAMEhead -n5 FILENAMEDisplay first 10 rows of file specified, or custom number of rows with -n#
ln TARGET /PATH/LINK_NAMEln -s TARGET /LINK/NAMECreate hard link to target in defined path, or create a symbolic link with -s (similar to standard Widnows link type)
User commandsAttributesDescription
susu - root, su rootSwitch user and move into user home directory with -, or just switch user and stay in working directory without -
sudo COMMAND---Run command as root, user needs to be in /etc/sudoers file (can be set with NOPASSWD: ALL command to not required password authentification
exit---Log out of current user
logout---Log out of current user in login shell
ssh USER_LOGIN@IP---SSH login to remote machine via username and IP
useradd USER_LOGINuseradd -c "COMMENT WITH SPACES" -d -e YYYY-MM-DD -g GROUP_ID_OR_NAME -m HOME_DIRECTORY_NAME -u USER_LOGINCreate new user - saved at /etc/passwd
userdel USER_LOGINuserdel -r USER_LOGINDelete defined user, with -r deltes also users home directory
usermod USER_LOGINusermod -c -d -e -g -m -u -s USER_LOGINModifies user data for user_login, similar as in useradd command
passwd USER_LOGIN---Change password for specified user
groupadd GROUP_NAMEgroupadd -g GROUP_ID GROUP_NAMEAdd new group to system, with -g add also unique group ID
chown OWNER:GROUP FILENAME---Change owner and group of file
chgrp GROUP FILENAME---Change group of file
alias NAME="VALUE"---Create alias for command
id USER_LOGIN---Shows id of specified or currently logged user
Text commandsAttributesDescription
vim FILENAME---Create or open file in vim text editor, commands in vim editor: i = insert mode, Esc = close insert mode, :!wq or ZZ = save and quit, :q! = quit without saving, dd = remove a line, yy = copies a line, p = paste line, u = undo, x = removes a character
cat FILENAMEcat -n FILENAMEShows file content in terminal window, with -n shows also line number
less FILENAME---Shows file content in terminal window in interactive format, able to search
cut FILENAMEcut -f NUMBER_OF_CELL -d DELIMITER_OF_CELLS FILENAMEShows content in terminal window, with -f shows specified cell, with -d change delimiter
vi, joe, emacs, nanoDe...---Another text editors
wc FILENAMEwc -lCount words in file, or count lines with -l
Other system commandsAttributesDescription
psps -efShows processes, -e for all processes, -f for full format
freefree -m, free -gShows ram memories, with -m in MB, with -g in GB
toptop -n1Shows processes in interactive table, or snapshot of table via -n1
kill PROCESS---Kill running process
yum update---Update OS
yum search PACKAGE, yum install PACKAGE, yum remove PACKAGE---Search, install or remove package via yum
apt search PACKAGE, apt-get install PACKAGE, apt remove PACKAGE---Search, install or remove package via apt
lsblklsblk -fShows hard drives installed, with -f shows also file system
mkfs.TYPE /dev/HDDmkfs -t TYPE /dev/HDDMake filesystem of defined type on defined harddrive
mount /dev/HDD /PATH_WHERE_TO_MOUNTmount -aTemporary mount of HDD, or remount all HDDs with -a
dfdf -hShow disk sizes, with -h shows in human readable format
fdisk /dev/HDD---Partitioning of defined hard drive
umount /PATH_WHERE_MOUNTED---Unmount HDD from defined folder
pvcreate /dev/HDD---Create tag LVM in physical volume PV HDD for attach it to volume group VG
vgcreate VG_NAME /dev/HDD1 /dev/HDD2---Create volume group VG from specified PV HDDs with LVM tag
lvcreate -n LV_HDD_NAME -L#SIZE /dev/VG_NAMEsee commandCreate logical volume LV with name -n, defined size -L from specified volume group
mkfs.TYPE /dev/VG_NAME/LV_NAME---Make filesystem of defined type on logical volume
pvs---Display informations about physical volumes
vgs---Display informations about volume groups
lvs---Display informations about logical volumes
lvremove, vgremove, pvremove---Commands for removing LVs, VGs, PVs
ifconfig---Shows network adapters with IPs
iptables -Fiptables -FFlush IP tables
systemctl status PACKAGE---Shows status of defined service
systemctl start PACKAGE, systemctl stop PACKAGE---Manually start or stop defined service
systemctl enable PACKAGE, systemctl disable PACKAGE, systemctl is-enabled PACKAGE---Enable or disable auto-starting of defined service or shows if enabled
tar cvzf ARCHIVE.tgz FILE_TO_ARCHIVE FILE2_TO_ARCHIVE---Compress specified files into archive via tar (can be done by command gzip)
tar tvf ARCHIVE.tgz---Show archive content
tar xvf ARCHIVE.tgz---Uncompress defined archive
File pathDecription

/etc/groupGroup list
/etc/shadowEncrypted password list
/etc/os-releaseOS informations
/etc/fstabHDD auto mounting file
/usr/bin/fishPath to fish terminal after insatlled by: yum install fish
/dev/sda-sdb-sdc-sddHard drives
/var/log/messagesSystem log
ExtrasDecription
#!/bin/bashFirst line in bash file, not located in bin folder
chmod 770 FILENAME7 = RWX, 6 = RW-, 5 = R-X, 4 = R--, 3 = -WX, 2 = -W-, 1 = --X, 0 = ---, or any other combination with a = all users, u = owner, g = group, o = others, + = add, - = remove, r = read, w = write and x = executable, for example a+rwx, u-x, go-wx
echo "text replace" > TEXT_FILEWrite terminal output to new file or replace old file
echo "text add" >> TEXT_FILEWrite terminal output to new file or add to old file
Ctrl+DCommand exit
TabComplete command, path or filename
UpArrowList previous used commands
cut -f 1 -d : /etc/passwdShows first row (usernames) of all users in /etc/passwd delimited by :
yum install tree -yInstall tree command via yum and do not ask for user confirmation
yum install httpd -yInstall http server via yum and do not ask for user confirmation
yum install fishInstall fish terminal via yum
usermod -s /usr/bin/fish USER_LOGINAdd fish terminal to specified user
find /START_DIRECTORY -iname "*STRING*"Find and file or folder containing STRING in name
find / -user USER_LOGINFind every file owned by defined user
uptimeShows details about system up time
ip a s | grep inetShows lines of IP command, where line starts with string inet
grep -i error /var/log/messagesShows all logs with error messages, not case sensitive
NetworkingDecription
ip a sDisplay Network Interfaces and IPs
ifconfigDisplay Network Interfaces and IPs
route -nDisplay network routing table
